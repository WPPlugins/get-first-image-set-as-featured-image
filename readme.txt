== Installation ==

This section describes how to install the plugin and get it working.

In WordPress:

    Go to Plugins > Add New > search for :: Get First Image Set As Featured Image ::
    Press Install Now for the Get First Image Set As Featured Image plugin
    Press Activate Plugin

To install manually instead:

    Upload the get_first_image_set_as_featured Image directory to the /wp-content/plugins/ directory
    Activate the plugin through the Plugins menu in WordPress
